Restest
=============================================================================================================================

---
Restest is a sample application, written in C++, for testing [Restbed](https://github.com/corvusoft/restbed).

Build
-----

```bash
git clone --recursive https://github.com/asmirnou/restest.git
mkdir restest/build
cd restest/build
cmake ..
make 
make test
```

Run
-----

```
./restest
```

Test
-----
[http://localhost:8080](http://localhost:8080)
